BGP VPN Configuration Files

This page provides direct access to Terraform configuration files for a BGP-based VPN setup on FortiGate firewalls, including primary and secondary tunnels connecting to an AWS environment.

Available Downloads

  • primary-vpn.tf – Primary VPN tunnel configuration
  • secondary-vpn.tf – Secondary (failover) VPN tunnel configuration
  • README.md – Full documentation and implementation notes
Note: All links point to the raw content hosted on GitHub. Clicking a file will trigger an immediate download instead of opening it in the browser. This page is optimized for GitHub Pages.
